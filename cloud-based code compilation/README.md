## Features
- **Web Playground:** Real-time HTML, CSS, and JS editor with live preview.
- **Admin Panel:** Manage users (add, edit, delete, enable/disable).
- **Activity Logs:** Monitor user login and logout times.
- **Multi-Language Support:** Python, JavaScript, C, C++.
- **Premium UI/UX:** Modern dark/light themes with glassmorphism.
- **User Authentication:** Email-based login with persistent sessions.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Access at: `http://localhost:5000`
